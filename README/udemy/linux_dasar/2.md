## nano
- shorcut
    - ctrl+x => exit
    - +o => save
    - +k => cut
    - +u => insert / paste    
    - +6 => maark
    - ctrl +w => search
        - +r => replace
    - ctrl +\ => replace
    - +/ => go to line

    - alt+shift+u => undo
    - alt+m => redo

```bash
nano -B file.txt # nano file with backup
    file.txt file.txt~

nano +3500,20 file.txt # position ur edit
```

## vim
- command editor vim ESC, :
    - w => write
    - q => quit
    - ! => force
    - 

- mode vim
    - i => insert
    - v => visual 
    - ESC => escape

    - R => replace
    - / => search
        - n => next
        - p => previous

- command vim
    - dd => delete line
    - 5dd => delete 5 line
    - 5dw => delete line with 2 column

    - p => paste
    - U => undo
    - 3u => undo 3

    - 3w => move next 3 text cursor
    - 3h => move previous 3 text cursor
    - 3j => move 3 line bottom
    - 5k => move 5 line top
